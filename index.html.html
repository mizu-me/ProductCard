카드의 이미지·배지·텍스트를 클릭해 직접 수정하세요. 카드마다 이미지(사진 업로드) · 링크(클릭 시 이동할 URL) · 배지(할인율) 버튼이 있습니다. 완성되면 상단 "이미지로 복사"로 Outlook에 붙여넣거나, 보관함 탭에서 저장해 다른 사람과 공유하세요.
아래 코드를 복사해 Outlook 서명/본문 HTML 소스에 붙여넣을 수 있습니다.
보관함에는 완성된 카드 이미지와 설정값이 함께 저장됩니다. "편집에 불러오기"로 편집 화면에 다시 불러와 수정하거나, 다른 사람이 만든 카드를 그대로 사용할 수 있습니다. 여러 사람과 함께 쓰려면 공유 / 엑셀 연동 탭에서 공유 파일을 연결하세요.
선택 0개

공유 저장소 연결 (같은 폴더/네트워크 드라이브)

보관함 데이터(카드 이미지+설정값)를 폴더 안의 JSON 파일 하나로 관리합니다. 이 파일을 팀 공유 폴더에 두면, 폴더에 접근할 수 있는 모든 사람이 같은 보관함을 보고 추가할 수 있습니다. (Chrome/Edge 브라우저에서 지원됩니다.)

연결된 공유 파일 없음

⚠️ 이 창을 새로고침하거나 다시 열면 "연결됨" 표시가 사라집니다. 그때마다 매번 새 파일을 만들 필요는 없고, "🔄 이전에 연결했던 파일로 재연결" 버튼(보이면) 또는 "② 기존 공유 파일 열기"로 같은 파일을 다시 선택하면 됩니다.

파일로 내보내기 / 가져오기 (모든 환경 지원)

공유 폴더 연결이 안 되는 환경(다른 브라우저 등)에서는 JSON 파일을 메일/메신저로 주고받아 보관함을 합칠 수 있습니다.

엑셀에서 상품 값 가져오기

견적서 양식의 "데이터" 시트처럼 품명 · 형번 · 규격 · 수량 · 단가 · 납기 · 비고 열이 있는 엑셀 파일을 불러와, 원하는 행을 클릭하면 새 카드가 자동으로 만들어집니다.

카드의 이미지·배지·텍스트를 클릭해 직접 수정하세요. 이미지는 3가지 방법으로 넣을 수 있습니다: ① 파일선택 버튼으로 PC 사진 업로드, ② 이미지URL 버튼에 kr.misumi-ec.com 상품페이지에서 복사한 이미지 주소 붙여넣기, ③ 이미지 칸을 클릭한 뒤 캡처한 화면을 그대로 Ctrl+V. 완성되면 상단 "이미지로 복사"로 Outlook에 붙여넣거나, 보관함 탭에서 저장해 다른 사람과 공유하세요.
아래 코드를 복사해 Outlook 서명/본문 HTML 소스에 붙여넣을 수 있습니다.
보관함에는 완성된 카드 이미지와 설정값이 함께 저장됩니다. "편집에 불러오기"로 편집 화면에 다시 불러와 수정하거나, 다른 사람이 만든 카드를 그대로 사용할 수 있습니다. 여러 사람과 함께 쓰려면 공유 / 엑셀 연동 탭에서 공유 파일을 연결하세요.
※ 형번/상품명의 앞부분 몇 글자만 입력해도 포함되는 카드가 전부 보입니다.

저장된 세트 (카드 여러 개를 이름 붙여 묶어두기)

지금 카드 편집 화면에 있는 카드들을 통째로 이름 붙여 저장해두고, 나중에 그 이름으로 한 번에 다시 불러올 수 있습니다. (예: "냉각기기 3종 세트")

"이미지로 복사" 또는 "Outlook 메일 준비"를 누른 순간마다 자동으로 기록됩니다 (실제로 메일을 보냈는지는 확인할 수 없지만, "이 고객에게 이 상품을 이때 준비해서 보냈다"를 추적하는 용도입니다). 같은 고객에게 중복으로 보내는 걸 방지하거나, 방문 전에 "저번에 뭘 보냈었지" 확인할 때 씁니다.
메일주소록에 등록했거나, 보관함/발송이력에 한 번이라도 이름이 나온 고객사가 왼쪽 목록에 모두 모입니다. 고객사를 클릭하면 그 고객에게 지금까지 보낸 이력과 저장해둔 카드가 한 화면에 나옵니다 — 방문 전 5분 브리핑용으로 쓰세요.
왼쪽에서 고객사를 선택하세요.
"Outlook 메일 준비" 버튼을 눌렀을 때 자동으로 채워지는 제목/본문 문구입니다. 아래 중괄호 토큰은 카드 내용에 맞춰 자동으로 바뀝니다: {고객사}(고객사명, 없으면 빈칸) · {대표상품}(첫 카드 상품명) · {외건수}(카드가 2개 이상이면 " 외 N건") · {상품목록}(카드별 상품명/형번/가격 줄바꿈 목록). 수정 후 "이 문구를 기본값으로 저장"을 누르면 다음부터 계속 이 문구가 쓰입니다.

TA_LIST 업로드 (고객사명 자동완성)

SQL에서 내려받은 TA_LIST(고객번호·고객명·SUB고객명·팀구분·셀코드·외근영업·내근영업·업계) CSV/엑셀 파일을 올리면, 상단 "고객사명" 칸에 입력할 때 목록에서 골라 쓸 수 있게 자동완성이 됩니다. 한 번 올려두면 다음에 다시 열 때도 그대로 남아있고, 최신 파일로 다시 올리면 갱신됩니다.

공유 저장소 연결 (같은 폴더/네트워크 드라이브)

보관함 데이터(카드 이미지+설정값)를 폴더 안의 JSON 파일 하나로 관리합니다. 이 파일을 팀 공유 폴더에 두면, 폴더에 접근할 수 있는 모든 사람이 같은 보관함을 보고 추가할 수 있습니다. (Chrome/Edge 브라우저에서 지원됩니다.)

연결된 공유 파일 없음

⚠️ 이 창을 새로고침하거나 다시 열면 "연결됨" 표시가 사라집니다. 그때마다 매번 새 파일을 만들 필요는 없고, "🔄 이전에 연결했던 파일로 재연결" 버튼(보이면) 또는 "② 기존 공유 파일 열기"로 같은 파일을 다시 선택하면 됩니다.

파일로 내보내기 / 가져오기 (모든 환경 지원)

공유 폴더 연결이 안 되는 환경(다른 브라우저 등)에서는 JSON 파일을 메일/메신저로 주고받아 보관함을 합칠 수 있습니다.
